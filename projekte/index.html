Meine persönliche Webseite:
Diese meine persönliche Webseite habe ich komplett von Hand ausschließlich in HTML, CSS und eine bisschen JavaScript geschrieben. Sie ist vollständig responsiv und sowohl auf großen Bildschirmen als auch auf kleinen Handys verwendbar. Angefangen habe ich damit in ende 2023, seit dem verbessere ich sie Vorlaufend. Wenn du wissen willst wie diese aufgebaut ist, dann schau dir gerne mal den Quelltext an, du findest ihn auf GitHub.

Meine Online-Projekt Webseite:
Ebenso wie meine persönliche Webseite habe ich meine Online-Projekt-Seite www.aocalux.com ebenfalls komplett von Hand, in HTML, CSS und JavaScript geschrieben. Mein Blog:
Mein Blog blog.chris-prickartz.de besteht ebenso nur aus HTML, CSS und JavaScript und ist bewusst sehr simpel gehalten. Die Struktur soll dabei übersichtlich sein, und den Fokus auf den Content legen, so das die Nutzende Person nicht zu stark vom eigentlichen Inhalt abgelenkt wird.
